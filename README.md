# 🚗 LojaCars

Landing page de uma loja de carros, desenvolvida com HTML, CSS e JavaScript puro. O projeto apresenta uma hero section com imagem de fundo, navegação responsiva e menu hamburguer para dispositivos móveis.

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização com variáveis CSS, Flexbox e Media Queries
- **JavaScript (Vanilla)** — lógica do menu hamburguer responsivo

## 📁 Estrutura de pastas

```
LojaDeCarros-main/
├── index.html
└── src/
    ├── assets/
    │   └── carro.jpg
    ├── css/
    │   └── estilo.css
    └── js/
        └── script.js
```

## ✨ Funcionalidades

- **Hero section** em tela cheia com imagem de fundo e overlay gradiente
- **Navbar** com links para Home, Carros, Lojas e Contato
- **Menu hamburguer** animado para telas com largura ≤ 1024px
- **Responsividade** via Media Queries
- **Scroll suave** configurado via CSS
- **Variáveis CSS** centralizando cores e estilos reutilizáveis

## 🚀 Como executar

1. Clone ou baixe o repositório:
   ```bash
   git clone https://github.com/seu-usuario/LojaDeCarros.git
   ```

2. Abra o arquivo `index.html` diretamente no navegador — não é necessário nenhum servidor ou instalação.

## 📱 Responsividade

| Dispositivo | Comportamento |
|---|---|
| Desktop (> 1024px) | Menu horizontal na navbar |
| Mobile/Tablet (≤ 1024px) | Menu hamburguer com slide lateral |
